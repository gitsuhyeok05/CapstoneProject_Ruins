# 캡스톤 디자인 프로젝트
## 절차적 생성을 활용한 배틀로얄게임 'Ruins'
한국공학대학교 게임공학과
김수혁, 배성주, 편완, 홍지창
유튜브 시연 링크 : https://www.youtube.com/watch?v=RaLPM1jvRVA

### 프로젝트 개요
게임 산업이 발전하고 커뮤니티가 확산됨에 따라 콘텐츠 소비 속도가 점점 빨라지고 있다. 
이에 발맞춰, 지난 1년간 우리는 콘텐츠 생산 속도 또한 높일 수 있는 방안을 모색해왔다. 
Ruins 게임에서는 절차적 생성 알고리즘을 활용하여 게임 맵과 플레이어 캐릭터의 걷기, 달리기와 같은 이동 모션을 구현하였고 실제 게임에 적용시켰다. 
향후에는 절차적 생성 기법의 성능을 향상시켜 콘텐츠 생산 효율을 높이는 것이 목표이다.
